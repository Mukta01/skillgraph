Instructions:
1. Analyze the list of missing skills and their categories/levels.
2. Order them logically — foundational skills before advanced ones, prerequisites before dependents.
3. Group them into 3-5 learning phases:
   - Phase 1: Foundations (essential prerequisites)
   - Phase 2: Core Skills (primary role requirements)
   - Phase 3: Advanced Skills (deeper expertise)
   - Phase 4: Specialization (nice-to-have, differentiators)
4. For each phase, estimate a realistic duration in weeks (assuming ~10-15 hours/week of study).
5. For each skill in each phase:
   - Estimate total hours to reach the required level.
   - Write a brief description of what to learn.
   - Suggest 2-3 FREE learning resources with real, working URLs. Prefer:
     * Official documentation (e.g., docs.python.org, react.dev)
     * YouTube channels (e.g., freeCodeCamp, Traversy Media, Khan Academy)
     * Free courses (e.g., Coursera audit, edX, freeCodeCamp, Khan Academy)
     * Interactive platforms (e.g., W3Schools, MDN Web Docs, Codecademy free tier)
   - Classify each resource type as: course, documentation, video, tutorial, or book.
6. Calculate the total weeks needed for the entire roadmap.

IMPORTANT:
- Be realistic with time estimates. Don't underestimate.
- Only suggest FREE resources. No paid courses.
- Resource URLs must be real and commonly known. If unsure of the exact URL, use the main domain (e.g., "https://www.freecodecamp.org").
- This must work for ANY field — technology, marketing, finance, healthcare, design, etc.
- If there are only 1-3 missing skills, it's fine to have fewer phases.
